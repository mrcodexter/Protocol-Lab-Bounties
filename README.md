🚀 PL_Genesis Hackathon Suite

A production-ready AI-powered Web3 project built for the PL_Genesis: Frontiers of Collaboration Hackathon.

This application combines modern AI capabilities with blockchain integration to deliver a scalable and innovative solution.

---

🔥 Features

- 🤖 AI-powered application (Google Gemini API)
- ⚛️ Built with React + TypeScript + Vite
- 🔗 Flow Blockchain Integration (Wallet + On-chain interaction)
- 🎨 Clean and responsive UI
- ⚡ Fast and lightweight architecture

---

🧠 Tech Stack

- Frontend: React, TypeScript, Vite
- Styling: CSS / Tailwind (if used)
- AI: Google Gemini API
- Blockchain: Flow (FCL - Flow Client Library)
